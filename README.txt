# Colonia's Finest – Website Starter

## Dateien
- index.html – Startseite
- style.css – Design und Responsive Layout
- logo.png – Hier eure vorhandene transparente Logo-Datei ablegen

## Verwendung
1. `index.html` und `style.css` in euer Vercel-Projekt übernehmen.
2. Eure bereits vorhandene transparente Logo-Datei als `logo.png` in denselben Ordner legen.
3. Deployen.

Die fünf Bereiche BAND, KONZERTE, GALERIE, SHOP und BOOKING sind zunächst Platzhalter und können im nächsten Schritt mit den echten Inhalten gefüllt werden.
